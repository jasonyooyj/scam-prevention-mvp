# 아이디어 브레인스토밍

## 후보 아이디어

### 1. URL/링크 실시간 검증 서비스
- 의심 URL 입력 시 피싱 여부 즉시 판별
- 웹사이트 유사도 분석, 도메인 등록 정보 확인
- 브라우저 확장 프로그램 형태 가능

### 2. 문자/메시지 피싱 탐지기
- 스미싱 문자 복사 붙여넣기로 분석
- 패턴 기반 + ML 기반 탐지
- 신고/공유 기능

### 3. 보이스피싱 음성 분석
- 통화 녹음 파일 분석
- 스캠 패턴 키워드/억양 탐지
- 실시간 경고 알림

### 4. 스캠 패턴 교육 챗봇
- 인터랙티브 피싱 시뮬레이션
- 최신 사기 수법 교육
- 퀴즈/게임화 요소

### 5. 의심 번호/계좌 조회 서비스
- 신고된 전화번호/계좌번호 DB 조회
- 커뮤니티 기반 신고 시스템
- 위험도 점수 제공

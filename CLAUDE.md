## 프로젝트 구조
```
scam-prevention-mvp/
├── src/
│   ├── app/
│   │   ├── page.tsx                # 메인 화면 (카테고리 선택)
│   │   ├── layout.tsx              # 루트 레이아웃

│   │       └── explanation/route.ts # AI 해설 API
│   ├── components/ui/              # shadcn/ui 컴포넌트
│   └── lib/
│       ├── db.ts                   # Neon 연결
│       ├── openai.ts               # OpenAI API
│       └── utils.ts                # 유틸리티
├── drizzle/
│   └── schema.ts                   # DB 스키마
└── .env.local                      # DATABASE_URL, OPENAI_API_KEY
```

## 개요

본 프로젝트의 퀴즈 데이터는 **실제 피싱·스캠 사례**를 기반으로 교육 목적에 맞게 재구성하였습니다. 모든 데이터는 공개된 정보를 기반으로 하며, 개인정보나 저작권을 침해하지 않습니다.

## 데이터 생성 원칙

### 1. 자체 생성 데이터
본 프로젝트의 퀴즈 콘텐츠는 다음 원칙에 따라 **자체 생성**되었습니다:

- 공개된 사기 패턴을 기반으로 **교육용 시나리오** 구성
- 실제 피해자 정보 또는 개인정보 **미포함**
- 사기 메시지의 특징을 학습할 수 있도록 **교육적 재구성**

### 2. 저작권 준수
- 정부·공공기관 자료: 공공누리 라이선스에 따른 활용
- 뉴스 보도: 보도 목적 인용 (저작권법 제28조)
- 기업 보안 안내: 공익 목적 참고 (비상업적 활용)

### 3. 개인정보 보호
- 모든 전화번호, 계좌번호, URL은 **가상의 정보**로 대체
- 실존 피해자 정보 미포함
- 실제 사기범 정보 미포함

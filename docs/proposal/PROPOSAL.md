# 탐정 안속아: 퀴즈형 피싱·스캠 예방 교육 서비스

> DACON 피싱·스캠 예방을 위한 서비스 개발 경진대회 MVP 기획서

---

## 1. 서비스 개요

### 1.1 서비스명
**탐정 안속아** (Scam Detective)

### 1.2 핵심 컨셉
실제 피싱·스캠 사례를 기반으로 한 **인터랙티브 퀴즈 게임**을 통해 사용자가 직접 사기 메시지를 판별하는 경험을 제공하는 예방 교육 서비스

### 1.3 타겟 사용자
- **주요 타겟**: MZ세대 (20~40대)
- **확장 타겟**: 디지털 금융 서비스를 이용하는 전 연령층

### 1.4 핵심 가치
> "직접 경험해봐야 안 속는다"

사기 메시지를 실제로 판별해보는 **체험형 학습**을 통해, 단순 경고나 홍보보다 높은 예방 효과 달성

---

## 2. 문제 정의

### 2.1 피싱·스캠 피해 현황

| 지표 | 수치 | 출처 |
|------|------|------|
| 보이스피싱 피해액 (2024) | 약 7,744억원 | 금융감독원 |
| 스미싱 피해 건수 (2024) | 50만건+ | KISA |
| MZ세대 피해 비율 | 전체의 42% | 경찰청 |

### 2.2 기존 예방 교육의 한계

1. **일방적 정보 전달**: 뉴스, 공익광고 등 수동적 콘텐츠
2. **현실감 부족**: 실제 사기 메시지와 동떨어진 예시
3. **학습 동기 부재**: 재미없고 지루한 교육 방식
4. **개인화 부재**: 모든 사용자에게 동일한 내용

### 2.3 해결해야 할 과제

> 어떻게 하면 사람들이 **자발적으로** 피싱·스캠 예방법을 학습하게 할 수 있을까?

## 3. 솔루션: 탐정 안속아

### 3.1 핵심 기능

#### 기능 1: 퀴즈 기반 사기 판별 게임
- 실제 사기 메시지 vs 정상 메시지 판별
- 4가지 카테고리: 스미싱, 중고거래 사기, 알바 사기, SNS 사기
- 총 104문항의 실전 시나리오

#### 기능 2: AI 해설 시스템
- OpenAI GPT-5.2 기반 개인화 해설
- 왜 사기인지/왜 안전한지 친근한 말투로 설명
- 사기 판별 포인트 학습 지원

#### 기능 3: 랭킹 & 경쟁 시스템
- 전체 사용자 순위표
- 카테고리별 랭킹
- 개인 통계 (정답률, 연속 정답 등)

#### 기능 4: 고급 모드 (HARD MODE)
- 정교하게 위장된 고난이도 시나리오 32문항
- 실제 피해자들도 속았던 정교한 수법

### 3.2 사용자 경험 플로우

```
[메인 화면]
     ↓
[카테고리 선택] → 스미싱 / 중고거래 / 알바 / SNS
     ↓
[퀴즈 플레이] → 메시지 읽기 → "사기" or "정상" 선택
     ↓
[결과 & AI 해설] → 정답 확인 + 상세 설명
     ↓
[최종 결과] → 점수 + 등급 + 랭킹 등록
```

### 3.3 차별화 포인트

| 구분 | 기존 서비스 | 탐정 안속아 |
|------|------------|-------------|
| 학습 방식 | 일방적 정보 전달 | 능동적 체험 학습 |
| 콘텐츠 | 단순 예시 | 104개 실제 사례 기반 |
| 피드백 | 없음 | AI 개인화 해설 |
| 동기 부여 | 없음 | 랭킹, 점수, 등급 시스템 |
| 난이도 | 단일 | 일반 + 고급 모드 |

### 4.4 AI 활용

```typescript
// AI 해설 생성 예시
async function generateExplanation(message: string, isScam: boolean) {
  const response = await openai.chat.completions.create({
    model: 'gpt-5.2',
    messages: [
      {
        role: 'system',
        content: '당신은 피싱/스캠 예방 전문가 "탐정 안속아"입니다.'
      },
      {
        role: 'user',
        content: `이 메시지가 ${isScam ? '사기' : '정상'}인 이유를 설명해주세요.`
      }
    ]
  });
  return response.choices[0].message.content;
}
```

## 5. 데이터 출처 및 수집

### 5.1 데이터 수집 원칙
- 공개된 사기 패턴 기반 **교육용 시나리오 자체 생성**
- 개인정보 및 실제 피해자 정보 **미포함**
- 저작권법 및 개인정보보호법 **준수**

### 5.2 참고 자료 출처

| 분류 | 기관/출처 | 건수 |
|------|----------|------|
| 정부·공공기관 | 금융감독원, 경찰청, KISA 등 | 8개 |
| 금융기관 | KB금융, 국민은행, 토스 등 | 5개 |
| 플랫폼 기업 | 네이버, 카카오, 당근마켓 등 | 8개 |
| 언론 보도 | YTN, SBS, 뉴스1 | 3개 |
| 소비자 보호 | 한국소비자원, 더치트 | 2개 |

> 상세 출처는 별도 문서 참조: [DATA_SOURCES.md](../DATA_SOURCES.md)

## 6. 사회적 가치 및 기대 효과

### 6.1 직접적 효과
- **체험 학습**을 통한 사기 판별 능력 향상
- **AI 해설**로 사기 수법 패턴 이해
- **반복 학습**을 통한 경각심 유지

### 6.2 간접적 효과
- 가족·지인에게 공유하여 **2차 예방 효과**
- 랭킹 시스템으로 **사회적 경쟁** 유도
- 새로운 사기 수법 **빠른 학습** 가능

### 6.3 정량적 기대 효과 (가정)

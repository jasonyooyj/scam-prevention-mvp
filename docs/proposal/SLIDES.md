## 슬라이드 2: 문제 정의

### 피싱·스캠 피해, 왜 줄지 않을까?

**피해 현황**
- 보이스피싱 피해액: **7,744억원** (2024)
- MZ세대 피해 비율: **42%**
- 스미싱 피해: **50만건+** (연간)

**기존 예방 교육의 한계**
| 문제점 | 현황 |
|--------|------|
| 일방적 전달 | 뉴스, 공익광고 → 수동적 |
| 현실감 부족 | 단순 예시 → 실전과 괴리 |
| 학습 동기 없음 | 재미없음 → 관심 저하 |

> **핵심 질문**: 어떻게 하면 자발적으로 학습하게 할 수 있을까?

## 슬라이드 3: 솔루션 소개

### "직접 경험해봐야 안 속는다"

**탐정 안속아 = 게이미피케이션 + AI**

```
┌─────────────────────────────────────────────┐
│                                             │
│    [실제 사기 메시지]                         │
│           ↓                                 │
│    "이거 사기야? 정상이야?"                   │
│           ↓                                 │
│    [선택] → [정답 확인] → [AI 해설]           │

│                                             │
└─────────────────────────────────────────────┘
```

**핵심 가치**
- 체험형 학습으로 판별 능력 향상
- AI 해설로 사기 패턴 학습
- 경쟁 시스템으로 재방문 유도

## 슬라이드 4: 주요 기능 (1/2)

### 기능 1: 퀴즈 기반 사기 판별

**4가지 카테고리**
| 카테고리 | 예시 | 문항 수 |
|----------|------|---------|
| 스미싱 | 택배 사칭, 기관 사칭 문자 | 27 |
| 중고거래 | 안전결제 위장, 가품 판매 | 25 |
| 알바 사기 | 고수익 알바, 선입금 요구 | 25 |
| SNS 사기 | 투자 유혹, 로맨스 스캠 | 27 |

**총 104문항** (일반 72 + 고급 32)

### 기능 2: AI 개인화 해설

- OpenAI GPT-5.2 기반
- 왜 사기인지/왜 안전한지 친근하게 설명
- 사기 판별 포인트 학습 지원

## PPT 제작 가이드

### 디자인 권장사항

1. **색상**: 네이비(#1e3a5f) + 주황(#f97316) 조합
2. **폰트**: 제목 - Pretendard Bold / 본문 - Pretendard Regular
3. **아이콘**: Lucide Icons 사용 (프로젝트와 일관성)

### 스크린샷 촬영 위치

1. 메인 화면: `/`
2. 카테고리 선택 후: `/quiz/smishing`
3. 퀴즈 진행 중: 문제 화면
4. 결과 화면: AI 해설 표시
5. 랭킹 화면: `/ranking`

### 권장 도구

- Canva (무료 템플릿)
- Google Slides
- PowerPoint
